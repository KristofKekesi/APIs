## Documentation

#### Get prime numbers between 1000 numbers
E.g. 3348000-3349000
```http
GET https://api.kekesi.dev/Prime/3/348000-349000.json
```
E.g. 8555000-8556000
```http
GET https://api.kekesi.dev/Prime/8/555000-556000.json
```
